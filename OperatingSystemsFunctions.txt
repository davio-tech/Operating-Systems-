One operating systems function is the processor management which determines how much time and power is spent on each computer process. For example when opening a word document it could cause other processes to run in the background of the computer, so the processor management determines the inportance of each one giving it enough power to run
File Management neatly organizes each file that runs the operating system. It allows for easy navigation to whichever specific file the user is looking for. It keeps track of the user access settings and status of every file.
Device Management manages the device communication to their respective drivers. It keeps track of each device thats connected to the system. It decides which process gets access to a certain device and for how long
